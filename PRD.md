# 금융 AI 에이전트 통합 PRD

## 1. 제품 개요

금융 및 주식 투자에 대한 전문 지식이 부족한 일반인들이 안정적으로 수익을 창출할 수 있도록 돕는 인공지능 기반의 금융 에이전트 서비스입니다. 사용자는 챗봇을 통해 주식 관련 기초 지식을 학습하고, 개인 상황에 최적화된 투자 추천을 받으며, 실시간 금융 정보를 얻을 수 있습니다. 또한 가상 주식 거래소를 통해 실제 돈을 투자하기 전에 안전하게 투자 감각을 익힐 수 있습니다.

## 2. 제품 비전

### 목적

경제, 주식 관련 상식이 없는 일반인이 수익성을 보장할 수 있는 금융 거래가 가능하도록 돕는 에이전트입니다. 이를 통해 사용자들은 금융 시장에 대한 막연한 두려움을 해소하고, 정보의 비대칭성으로 인한 투자 실패 위험을 줄여 스스로 현명한 투자 결정을 내릴 수 있게 됩니다.

### 해결하려는 문제

- 주식 투자에 대한 정보 부족과 잘못된 정보 습득으로 인한 투자 손실
- 복잡한 금융 용어와 방대한 정보 속에서 자신에게 맞는 투자 전략을 세우기 어려움
- 실제 투자 전 연습할 수 있는 안전한 환경 부재
- 투자 결정 과정과 감정을 기록하고 복기할 수 있는 도구 부재

## 3. 기존 기능 (유지 및 개선)

### 3.1 사용자 성향 분석 설문조사

**기능:** 시작 시 설문으로 사용자 프로필 생성

**측정 항목:**
  - 충동성 (impulsivity): 투자 결정 스타일
  - 금융 관심도 (financialInterest): 금융 시장 관심 수준
  - 금융 이해도 (comprehension): 금융 용어 이해 수준

**개인화:** 설문 결과에 따라 AI 에이전트의 응답 스타일과 조언 방식이 개인화됨

### 3.2 대시보드

- 총 자산 현황 표시
- 최근 거래 내역 요약
- 자산 구성 차트 (파이 차트)
- AI가 생성한 개인화된 환영 메시지
- 사용자 수준에 맞는 금융 학습 팁 카드

**개선 사항:**
- 가상 거래소 수익률 통합 표시
- 실시간 뉴스 미리보기
- 최근 투자 일기 요약

### 3.3 AI 채팅 에이전트

**기존 기능:**
- Google Gemini API 기반 대화형 금융 상담
- 사용자 성향에 맞춘 개인화된 조언 제공
- 음성 합성(TTS) 지원 (한국어 음성 재생)
- 함수 호출 기능:
  - `getAssetSummary`: 자산 포트폴리오 조회
  - `getTransactionHistory`: 거래 내역 조회
  - `getStockPrice`: 주식 가격 조회

**개선 사항:**
- 가상 거래소 데이터 연동 (가상 포트폴리오 조회)
- 투자 일기 연동 (과거 투자 결정 검토)
- 실시간 뉴스 및 리포트 정보 제공
- 금융 용어 설명 기능 강화

### 3.4 거래 내역

- 전체/수입/고정 지출/변동 지출 필터링
- 날짜순 정렬된 거래 목록 표시

### 3.5 주식 시장

- 보유 주식 포트폴리오 시각화 (파이 차트)
- 각 주식의 평가금액, 수량, 가격 정보 카드 표시

### 3.6 외환 시장

- USD/KRW, EUR/KRW, JPY/KRW 환율 조회
- 환율 변동 차트 (라인 차트)
- 탭으로 통화 쌍 전환

### 3.7 다크 모드 지원

- 라이트/다크 모드 전환
- 시스템 설정 자동 감지

## 4. 추가되는 핵심 기능

### 4.1 로그인 및 회원가입

**사용자 가치:** 사용자 데이터를 안전하게 관리하고 개인화된 서비스를 제공받기 위해 필요합니다.

**상세 사양:**
- 이메일/비밀번호 기반 로그인 및 회원가입
- 간편 로그인 (카카오, 네이버 등 소셜 로그인 연동) - UI만 구현, 실제 연동은 향후 진행
- 비밀번호 찾기/재설정 기능 (향후 구현)
- 로그인 상태 유지 (localStorage 활용)

**구현 우선순위:** 높음

### 4.2 사용자 정보 입력 확장

**사용자 가치:** 사용자의 개인 정보와 투자 성향을 파악하여 챗봇이 더욱 정확하고 맞춤화된 조언을 제공할 수 있도록 합니다.

**상세 사양:**
- **기본 정보 입력 (회원가입 시 또는 첫 로그인 시):**
  - 나이, 성별, 직업, 소득 수준 등 기본 인적 사항
- **투자 성향 설문 (기존 설문 확장):**
  - 투자 경험, 투자 목표(수익성/안정성), 투자 기간 등 투자 성향 설문
  - 생활 패턴 (소비 습관, 저축액 등) 입력
- **기존 설문 유지:**
  - 충동성, 금융 관심도, 금융 이해도 설문

**구현 우선순위:** 높음

### 4.3 실시간 금융 정보 제공 확장

**사용자 가치:** 시장의 흐름을 빠르게 파악하고 적시에 투자 결정을 내릴 수 있도록 돕습니다.

**상세 사양:**
- **실시간 주가 (기존 기능 유지 및 개선):**
  - 사용자가 관심 있는 종목의 현재가, 등락률, 거래량 등을 실시간으로 표시
  - 주가 그래프: 일봉, 주봉, 월봉 등 다양한 기간별 주가 차트 제공
- **주요 뉴스 (신규 추가):**
  - 국내외 경제 및 증권 관련 속보를 실시간으로 업데이트
  - 뉴스 카테고리별 필터링 (시장 동향, 기업 뉴스, 경제 정책 등)
  - 관련 주식 종목 자동 연동
- **증권사 리포트 (신규 추가):**
  - 주요 증권사에서 발행하는 기업 분석 및 시장 전망 리포트 열람
  - 리포트 검색 및 필터링 기능
  - 투자의견 (매수/보유/매도) 및 목표가 표시

**구현 우선순위:** 중간

### 4.4 가상 주식 거래소 서비스

**사용자 가치:** 실제 돈을 들이지 않고 주식 거래를 연습하며 투자 감각을 익히고 전략을 시험해볼 수 있습니다.

**상세 사양:**
- **가상 투자 자금:** 초기 자본금 1억 원 제공 (가상)
- **실시간 주가 반영:** 실제 주가를 기반으로 한 매수/매도 기능
- **매수/매도 기능:**
  - 주식 종목 검색 및 선택
  - 수량 입력 및 주문 확인
  - 매수/매도 주문 실행
  - 주문 후 잔고 및 보유 종목 업데이트
- **가상 계좌 관리:**
  - 가상 계좌 잔고 및 보유 종목 현황 표시
  - 보유 종목별 평가손익 및 수익률 표시
  - 평균 매수가, 현재가, 총 평가금액 표시
- **투자 수익률 분석:**
  - 가상 투자 수익률 및 손익 분석 리포트
  - 기간별 자산 추이 차트
  - 종목별 수익률 분석
- **시장 상황 시뮬레이션:**
  - 폭락장, 약세장, 조정장 등 다양한 시장 상황 시뮬레이션 (향후 구현)
  - 특정 상황 발생 시 AI가 대처 방법 제안 (향후 구현)

**구현 우선순위:** 높음

### 4.5 투자 일기/메모 기능

**사용자 가치:** 자신의 투자 결정 과정과 감정을 기록하여 복기하고, 투자 원칙을 세우는 데 도움을 받습니다.

**상세 사양:**
- **투자 기록 작성:**
  - 종목별 또는 날짜별 투자 기록 작성
  - 투자 결정 근거 및 사유 기록
  - 감정 상태 기록 (흥분, 불안, 기대, 후회, 자신감, 평온 등)
- **메모 관리:**
  - 개인적인 메모 및 인사이트 기록
  - 태그 기능으로 메모 분류
  - 일기 수정 및 삭제 기능
  - 날짜별 정렬 및 조회

**구현 우선순위:** 중간

### 4.6 개인화된 알림 서비스

**사용자 가치:** 사용자가 설정한 조건에 맞춰 실시간으로 알림을 받아 중요한 정보를 놓치지 않도록 합니다.

**상세 사양:**
- **관심 종목 알림:**
  - 관심 종목 설정 및 목표 주가 도달 알림
  - 종목 가격 변동 알림 (설정 임계값 이상 변동 시)
- **시장 알림:**
  - 주요 경제 지표 발표 알림
  - 시장 변동성 확대 시 푸시 알림
- **챗봇 추천 알림:**
  - 챗봇 추천 종목 관련 중요 업데이트 알림
- **가상 거래소 알림:**
  - 매수/매도 주문 체결 알림
  - 손익 한도 도달 알림

**구현 우선순위:** 낮음 (기본 기능 우선 구현, 향후 확장)

## 5. 사용자 시나리오

### 시나리오 1: 초보 투자자 민수 (23세, 대학생)

민수는 주식을 통해 수익을 올리고 싶지만 문외한이라 주변의 잘못된 정보를 습득하고 오판을 하곤 했다. 그래서 적자를 보는 경우가 많아서 이번에는 AI 에이전트를 통해 수익성을 올리고자 했다.

1. **회원가입 및 정보 입력:** 이메일로 회원가입 후 기본 정보(나이, 직업 등)와 투자 성향 설문을 완료
2. **기초 지식 학습:** 챗봇을 통해 "ETF가 뭐야?", "PER이란?" 등 금융 기초 지식을 학습
3. **가상 거래소 연습:** 가상 주식 거래소에서 1억 원의 가상 자금으로 주식 거래를 연습하며 주식 거래의 흐름을 익힘
4. **실시간 정보 활용:** 실시간 주가 그래프를 참고하며 뉴스와 리포트를 읽으며 시장을 분석
5. **투자 일기 기록:** 가상 거래에서 얻은 인사이트와 감정을 투자 일기에 기록
6. **실전 투자 준비:** 자신의 목표(안정성 > 수익성)를 챗봇에 입력하여 적절한 주식을 추천받고 실거래 준비

### 시나리오 2: 주부 지영 (30대 후반)

지영은 육아와 가사를 병행하며 투자에 할애할 시간이 부족하다. 가정 경제에 보탬이 되고 싶지만 주식 용어가 어렵고 복잡하게 느껴져 시작하기를 망설이고 있다.

1. **간편 회원가입:** 카카오 간편 로그인으로 빠르게 가입
2. **맞춤형 설명 받기:** 챗봇이 금융 이해도가 낮은 사용자임을 파악하여 쉬운 언어로 금융 용어를 설명
3. **가상 거래소 체험:** 실제 돈 없이 안전하게 가상 거래소에서 투자를 체험하며 두려움 해소
4. **투자 일기로 학습:** 투자 결정 과정을 일기에 기록하며 투자 원칙을 세움
5. **알림 활용:** 관심 종목을 설정하고 목표 가격 도달 시 알림을 받아 빠르게 시장 상황 파악 (향후 구현)

## 6. 기술 스택

### 6.1 프론트엔드 (기존 유지 및 확장)

- **React.js + TypeScript:** 컴포넌트 기반 UI 구축
- **Vite:** 빠른 개발 환경 및 빌드
- **Tailwind CSS:** 반응형 UI 스타일링
- **Recharts:** 금융 데이터 시각화 (기존 사용 중)

### 6.2 백엔드 (향후 구현)

- **Node.js (Express.js):** API 서버 (현재는 프론트엔드만 구현)
- **Python (FastAPI):** AI/ML 모델 연동 (향후)

### 6.3 데이터베이스 (향후 구현)

- **PostgreSQL:** 사용자 정보, 거래 내역 등 정형 데이터
- **Redis:** 실시간 주가 캐시, 세션 관리

### 6.4 AI/ML (기존 유지)

- **Google Gemini API:** 챗봇 엔진 (gemini-2.5-flash)
- **TTS:** 한국어 음성 합성 (gemini-2.5-flash-preview-tts)

### 6.5 데이터 소스 (현재 Mock 데이터, 향후 실제 API 연동)

- **주가 데이터:** 한국투자증권 API 또는 유사 서비스
- **뉴스 데이터:** 뉴스 API 또는 크롤링
- **증권사 리포트:** 각 증권사 API

## 10. 참고 서비스

### 10.1 카카오증권 / 영웅문 S

**참고 내용:** 실시간 주가, 그래프, 뉴스, 증권사 리포트 등 금융 정보의 시각화 및 제공 방식

**이유:** 사용자들이 복잡한 금융 데이터를 한눈에 이해하기 쉽도록 직관적인 UI/UX를 제공하는 방법을 벤치마킹합니다. 특히 실시간 데이터의 가독성과 업데이트 속도에 중점을 둡니다.

### 10.2 콴다(QANDA) / 뤼튼 (Wrtn)

**참고 내용:** 대화형 AI 챗봇의 사용자 친화적인 인터페이스와 정보 전달 방식

**이유:** 챗봇을 통해 주식 관련 기초 지식을 설명하고, 개인화된 투자 상담을 제공하는 데 있어 자연어 처리 기술과 사용자 경험을 개선하는 데 참고합니다. 복잡한 내용을 쉽게 풀어 설명하는 방식이 중요합니다.

### 10.3 모의투자 서비스 (각 증권사 제공)

**참고 내용:** 가상 주식 거래 시스템의 기능 및 사용자 경험

**이유:** 실제 투자와 유사한 환경에서 사용자가 부담 없이 주식 거래 연습을 할 수 있도록 모의투자 시스템의 핵심 기능(매수/매도, 잔고 관리, 수익률 확인 등)을 참고하여 구현합니다. 또한 현실에서 발생할 수 있는 주식의 등락폭 및 실시간 주가 변화(폭락장, 약세장, 조정장, 투매, 데드캣 바운스, 하방 입력 등)를 의도적으로 가상으로 구현하여, 사용자가 여러가지 상황을 직접 체험할 수 있게 유도합니다. 사용자가 해당 상황에 대한 대처 방법을 질문한다면 서비스는 적절한 투자 방향을 직관적으로 제안해야 합니다.
